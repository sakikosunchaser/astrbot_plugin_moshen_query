魔裁群查询插件

一个用于 AstrBot 的魔裁群查询插件，支持按地区名称或简称查询对应的 QQ 群号。

## 功能

- 支持 `/魔裁查询`
  - 返回全部地区群号
- 支持 `/魔裁查询 北京`
  - 查询指定地区群号
- 支持 `/魔裁查询 京`
  - 支持地区简称查询

## 支持的地区

- 北京
- 辽宁沈阳
- 江浙沪
- 山西
- 西南
- 广东
- 川渝

## 使用示例

```text
/魔裁查询
/魔裁查询 北京
/魔裁查询 京
/魔裁查询 粤
/魔裁查询 沪
```
